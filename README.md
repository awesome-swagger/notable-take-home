## Getting Started

### Prerequisites

_If you didn't install npm on your local, you need to install it._

  ```sh
  npm install npm@latest -g
  ```

### Installation

_Below is how to install and set up this app._

1. Clone or download the repo

2. Run Back-end
    - ```sh
      cd back-end
      ```

    - ```sh
      npm install
      ```

      If you use `yarn`

      ```sh
        yarn
      ```
      
    - ```sh
      npm start
      ```

      If you use `yarn`

      ```sh
        yarn start
      ```

3. Run Front-end
    - ```sh
      cd front-end
      ```

    - ```sh
      npm install
      ```

      If you use `yarn`

      ```sh
      yarn
      ```

    - ```sh
      npm start
      ```

      If you use `yarn`

      ```sh
      yarn start
      ```
